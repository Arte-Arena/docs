Bem vindo à Documentação do Space!

Space (/speɪs/) é um ERP para gestão da empresa Arte Arena. Usando principalmente as linguagens PHP e Typescript, Space é um software de gestão empresarial que ajuda as empresas Arte Arena a gerenciar seus produtos, pedidos, vendas, clientes, fornecedores, produção, contas etc.

Space oferece uma API intuitiva e um conjunto de funcionalidades prontas para uso por todos os setores da empresa, oferecendo uma especial interface para acompanhamento dos pedidos, desde a venda até a entrega, passando pela arte-final, bem como pela impressão, confecção e demais etapas produtivas.

Nota

This project is under active development.
